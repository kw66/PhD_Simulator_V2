## ✨ 项目简介

《研究生模拟器 v2.0》是一个以中国研究生科研与校园生活为背景的浏览器模拟游戏。<br>
你需要按月安排科研与生活，在论文、导师要求、人际关系、经济状态和身心健康之间维持平衡，并让不同角色走出各自的成长路线。

v2.0 不是在旧代码上继续堆叠功能，而是使用 **Vite + TypeScript** 从规则、状态、持久化和界面层重新搭建。项目目前仍在持续开发，已经具备可游玩的主流程与自动化测试，但部分角色机制、数值平衡和内容仍会继续调整。

## 🎮 核心玩法

- 以月份为回合推进研究生阶段，在关键时间节点处理固定与随机事件。
- 从不同角色与导师组合开始游戏，面对差异化的属性、毕业要求和成长路线。
- 在以下资源之间持续取舍：
  - 🧠 SAN 与健康状态
  - 🔬 科研能力、论文进度与科研分
  - 👥 社交能力、同门关系与合作机会
  - ❤️ 导师好感与培养资源
  - 💰 金钱、商店物品与长期增益
- 通过论文投稿、学术会议、实习、联合培养和关系事件，逐步解锁新的选择与结局。

## 🤖 AI 开发说明

这个版本延续了 v1.0 的 AI 辅助开发方式，但不再把功能集中在少量大型脚本中。

- 先审计 [v1.0](https://github.com/kw66/PhD_Simulator) 的真实规则，再按低耦合结构重新实现。
- 游戏状态、规则、持久化和 UI 分层组织，方便独立修改或替换系统。
- 每轮重构都通过类型检查与自动化测试回查，减少复杂事件链中的行为回归。
- 当前主要使用 Codex、Claude 与 Gemini 协作完成规则审计、实现、测试和视觉迭代。
